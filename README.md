## task7_2.py — Практическое задание (поиск подстроки без повторений)

Скрипт работает со списком произвольных элементов:
1) Находит **самую длинную строку** в списке (если таких несколько — берёт **первую**).
2) В этой строке находит **самую длинную подстроку без повторяющихся символов**
   (если таких несколько — возвращает **первую**).

Если в списке **нет строк**, результат: индекс `-1` и пустая строка.

### Функции
- `check_input(element)` — возвращает `True`, если `element` является строкой, иначе `False`.
- `get_substring(s)` — возвращает самую длинную подстроку без повторений в строке `s`.
- `main(arr)` — находит индекс самой длинной строки в `arr`, затем выводит:
  `index; 'substring'` и возвращает `(index, substring)`.

### Пример
Вход:
```python
[[0, 4, 8], "abca", 7, "aaaaa"]
